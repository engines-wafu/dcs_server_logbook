Joint Strike Wing Branding Guidelines

Logo

The Joint Strike Wing logo is a core element of our identity. Use the logo as it appears here; do not modify or alter the logo in any way. The logo should not be compressed, stretched, or skewed. Always maintain a minimum clear space around the logo, equivalent to the height of the 'W' in the logo, to ensure its visibility and impact.

Color Palette

Our color palette is a key component of our brand identity. Use the following colors consistently across all media.

Typography

The typography system for the Joint Strike Wing gaming group uses three typefaces: Ungap, Benelux Medium, and Perpetua Std Titling Light.

Headings

Ungap should be used for all headings. It provides a strong, clear presence that supports our commanding brand voice.

Body Text

Benelux Medium and Medium Oblique should be used for all body text. They offer excellent readability and versatility in both digital and print formats.

Accent Typeface

Perpetua Std Titling Light is to be used sparingly as an accent typeface for pull quotes, callouts, or for specific navigational elements where a touch of elegance is required.

Usage of Typefaces

Our typefaces should be used in a way that maintains legibility and hierarchy. Always ensure that text is easily readable on any background color. Avoid using the accent typeface for lengthy text.

Headings (H1, H2, H3, etc.)

Use 'Ungap' for all headers. Maintain adequate spacing between letters and ensure contrast with the background.

Paragraphs and Body Text

For paragraphs, use 'Benelux Medium'. For emphasized text within paragraphs, use 'Benelux Medium Oblique'.

Accent Text

Use 'Perpetua Std Titling Light' for accentuating specific text elements such as quotes, important notices, or calls to action. This typeface should be used at a larger size to create impact.

Web and Digital Use

When applying our branding guidelines to web and digital platforms, ensure that all elements are optimized for screens. This includes retina displays and various device sizes. Use SVG formats for the logo to ensure it scales without loss of quality. Colors may display differently on screens; always check for color consistency across different devices.

Responsive Logo

The logo should remain clear and legible at all scales. Provide alternative logo arrangements for small screen sizes if necessary.

Color Contrast and Accessibility

Ensure there is sufficient color contrast between text and background colors, keeping accessibility in mind. Text should be easily readable by everyone, including individuals with visual impairments.

Typography on Web

Ensure that the typefaces used are legible on screens. Fonts should be embedded correctly and fallback fonts should be specified in CSS. For Ungap and Benelux Medium, ensure that web font versions are used to prevent any rendering issues.
